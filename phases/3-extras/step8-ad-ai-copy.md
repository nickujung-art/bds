# Step 8 (3-extras): ad-ai-copy

## 목표
광고주가 캠페인 카피를 작성할 때 Claude API로 카피 제안·표시광고법 준수 여부 감지를 제공한다.

## 전제 (Prerequisites)
- 1-launch step8 완료 (ad-admin)
- 3-extras step6 완료 (ai-chat — Claude API 인프라 재사용)

## 적용 범위 (Scope)
- `src/lib/ai/ad-copy.ts` — 카피 생성 + 표시광고법 감지
- `src/app/(admin)/admin/ads/copy-assist/` — 광고주 카피 작성 UI

## 기능 1: 카피 제안

입력: 캠페인 유형(분양/중개), 단지명, 평형, 주요 강점(자유 입력)
출력: 카피 3가지 버전 (30자 이내 헤드라인 + 60자 이내 본문)

시스템 프롬프트에 표시광고법 규정 포함:
- 분양 광고: 사업주체·입주예정일·분양보증 명시 의무
- "최고", "1위", "독보적" 등 최상급 표현 금지 (비교광고 기준 위반 위험)

## 기능 2: 표시광고법 준수 감지

광고주가 직접 입력한 카피를 Claude로 검토:
- 위반 의심 표현 탐지 → 경고 메시지 + 수정 제안
- 통과 시 "법무 검토 권장" 배너 (AI 감지는 보조 수단, 법무 최종 판단)

## 도메인 컨텍스트 / 가드레일
- **LLM 거짓 음성 책임 모델**: AI 감지 통과가 법적 보장이 아님을 UI에 명시. 광고주 동의 체크박스 필수 ("표시광고법 준수 여부는 법무사 확인 책임")
- 모델: `claude-haiku-4-5-20251001` (카피 생성). 법무 감지는 `claude-sonnet-4-6` (정확도 우선)
- 비용: step6 `ai_usage_logs` 재사용. 광고주당 일 10회 제한
- 생성된 카피는 DB 저장 안 함 (임시 렌더만). 광고주가 직접 복사·편집 후 캠페인 등록

## 작업 목록
1. `ad-copy.ts`: `suggestCopy(input)` + `checkCompliance(copyText)` 함수
2. 광고주 카피 작성 UI (캠페인 등록 폼 내 탭): "AI 카피 제안" + "법무 감지"
3. 준수 감지 결과: 통과(초록) / 위반 의심(노랑) / 위반 가능성 높음(빨강) 3단계
4. "AI 감지는 보조 수단" 면책 문구 + 동의 체크박스
5. Vitest: 최상급 표현 탐지 케이스 + rate limit

## 수용 기준 (Acceptance Criteria)

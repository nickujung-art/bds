`.github/workflows/molit-backfill-once.yml` 파일을 아래 내용으로 생성한다.

핵심 요구사항:
- `workflow_dispatch`만 사용 (자동 실행 없음 — 1회성)
- `inputs.sgg_codes`: 운영자가 지역 코드를 직접 입력
- MOLIT API 일 10,000회 한도 → `timeout-minutes: 300` (5시간)
- `--resume` 플래그 필수 → 중단 후 재개 가능
- 환경 변수: `MOLIT_API_KEY`, `NEXT_PUBLIC_SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY`
- `actions/checkout@v4`, `actions/setup-node@v4` (node-version: '20')

파일 내용:
```yaml
name: MOLIT Backfill (1회성)

on:
  workflow_dispatch:
    inputs:
      sgg_codes:
        description: 'SGG 코드 (쉼표 구분, 예: 48121,48123,48125). regions 테이블에서 확인.'
        required: true
        default: '48121,48123'

jobs:
  backfill:
    name: MOLIT 실거래가 백필
    runs-on: ubuntu-latest
    timeout-minutes: 300

    steps:
      - uses: actions/checkout@v4

      - uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'npm'

      - name: Install dependencies
        run: npm ci

      - name: Run MOLIT backfill
        env:
          MOLIT_API_KEY: ${{ secrets.MOLIT_API_KEY }}
          NEXT_PUBLIC_SUPABASE_URL: ${{ secrets.NEXT_PUBLIC_SUPABASE_URL }}
          SUPABASE_SERVICE_ROLE_KEY: ${{ secrets.SUPABASE_SERVICE_ROLE_KEY }}
        run: |
          echo "SGG codes: ${{ inputs.sgg_codes }}"
          npx tsx scripts/backfill-realprice.ts \
            --resume \
            --sgg=${{ inputs.sgg_codes }}

      - name: Backfill complete
        run: echo "백필 완료 — ingest_runs 테이블에서 결과 확인"
```

**실행 안내 주석은 파일에 포함하지 않는다.** 주석은 이 PLAN에만 기록한다.

실행 방법 (운영자용):
1. GitHub → Actions → "MOLIT Backfill (1회성)" → Run workflow
2. sgg_codes 입력 (예: 48121,48123,48125) → 하루 3-5개 sgg씩 분할
3. 완료 후 다음 날 다른 sgg_codes로 재실행 (--resume이 완료된 것 skip)
  </action>
